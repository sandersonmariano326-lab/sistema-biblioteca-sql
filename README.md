# Sistema Biblioteca SQL

Projeto criado para praticar SQL e montar um portfólio no GitHub.

## Objetivo

Aprender os principais comandos SQL por meio de um sistema simples de biblioteca.

## Arquivos do projeto

- criar_tabelas.sql
- inserir_dados.sql
- consultas.sql

## Comandos utilizados

- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT
- SELECT
- WHERE
- ORDER BY
- INNER JOIN

## Tecnologias

- SQL
- SQLite
- Git
- GitHub
